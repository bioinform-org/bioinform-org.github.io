{{ "BioINForming 2.0" }}

Participants: 18 high school students (8 in-person and 10 online)

Trainers: Katarina Elez, Selle Bandstra and Marco Anteghini

Program

1. Introduction
  • Introduction to bioinformatics
  • Basics of molecular biology and genetics
  • Basics of Unix OS
2. Molecular sequence analysis
  • Next Generation Sequencing (NGS)
  • Pairwise sequence alignment
  • Sequence database search
  • Multiple sequence alignment
3. Omics
  • Introduction to different omics
  • Differential gene expression
  • DNA-based species identification

* Career and Erasmus+ opportunities


4. Protein structure analysis
  • Structure determination
  • Basics of protein structure
  • Protein structure prediction
  • Protein structure visualization, comparison and classification
5. Bio(medical) image analysis
  • Introduction to bio(medical) image analysis
  • Basics of Python
  • Basics of bioimage analysis
